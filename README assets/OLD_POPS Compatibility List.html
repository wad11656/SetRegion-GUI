PSX to PSP GLOSSARY

Pops = The in-built PSX emulator for the PSP, developed by Sony. Each firmware has it’s own, though some share the same binary.

Popsloader = A plugin use to load different versions of pops.

Keys.bin = When you download a PSOne Classic form the PSN store, it comes with keys.bin and it’s required for the game to run. The old popstations generate one, but this is not required anymore with the newer popstations.

Document.dat = The game manual, which also comes PSOne Classics from the playstation store. You can make your own custom game manual with PSP DocMaker GUI.

GAME ID = The identification number used by Sony. Typically this is on the Playstation CD in a file named with the region code plus the unique number for each disk. For example, SLUS-00972 tells you it was shipped to the United States.

PSOne Classic = Also referred to as “PSN versions”, these PSX games available for purchase via the Playstation NetworkStore. These versions don’t require Popsloader, most work fine as long as you’re on the latest firmware.
